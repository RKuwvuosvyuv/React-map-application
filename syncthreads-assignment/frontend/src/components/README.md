# Syncthreads React + Express Assignment

## Project Setup

### 1. Backend Setup
```sh
cd backend
npm install
npm start
```

### 2. Frontend Setup
```sh
cd frontend
npm install
npm start
```

### 3. Technologies Used
- **React** for frontend UI
- **Leaflet.js** for OpenStreetMap integration
- **Express.js** for backend API handling
- **JWT Authentication** for secure login
- **Material UI** for modern UI styling

### 4. Deployment
- **Backend** can be deployed on Heroku/AWS/GCP.
- **Frontend** can be deployed on Netlify/Vercel.

### 5. Features
✅ Secure **JWT Authentication**  
✅ OpenStreetMap Integration (**Leaflet.js**)  
✅ **Material UI** for better UI/UX  
✅ **Protected Routes** (Dashboard & Map)
